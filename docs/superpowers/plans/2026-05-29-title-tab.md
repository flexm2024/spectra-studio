# 타이틀 탭 구현 계획

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Step2 비주얼 편집기 우측 패널에 "타이틀" 탭을 추가해 기본 스타일(9종), 데코 스타일(7종), 폰트(17종), 위치, 크기를 실시간으로 스테이지 프리뷰에 반영한다.

**Architecture:** Typography 인터페이스를 확장하고 CSS class 조합(`title-style-*`, `title-deco-*`)으로 스타일을 렌더링한다. 우측 패널 헤더를 탭 스위처로 교체해 기존 효과 UI를 유지하면서 새 타이틀 탭 공간을 확보한다.

**Tech Stack:** React 18, TypeScript, CSS custom properties (var(--c), var(--line) 등), Google Fonts API, Vitest + React Testing Library
